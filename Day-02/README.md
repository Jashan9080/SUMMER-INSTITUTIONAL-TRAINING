## Topics Covered
### 1. Introduction to Linux
- Linux is a free and open-source operating system.
- It is secure, stable and widely used in servers, cloud computing and software development.
- Linux supports multi-user and multitasking environments.
- Different Linux distributions include Ubuntu, Fedora, Debian, Kali Linux and Linux Mint.
### 2. Linux Shell
The Shell is a command-line interpreter that acts as an interface between the user and the Linux Kernel. It accepts user commands, processes them and displays the output.
#### Functions of Shell
- Accepts user commands.
- Executes Linux commands.
- Provides communication with the Kernel.
- Supports shell scripting.
- Automates repetitive tasks
### 3. Bash (Bourne Again Shell)
Bash is the default shell in most Linux distributions including Ubuntu. It provides an easy-to-use command-line environment for executing commands and writing shell scripts.
#### Features
- Command History
- Auto-completion
- Shell Scripting
- Variables and Aliases
- Easy command execution
### 4. Linux Command Line
The Linux Command Line (Terminal) allows users to interact with the operating system by typing commands instead of using a graphical interface.
#### Advantages
- Fast execution
- Better system control
- Automation through scripting
- Useful for system administration
- Requires fewer system resources
### Practical Performed
Linux was installed and executed using **Oracle VirtualBox**.
The following basic commands were practiced:
| Command | Purpose |
| `date` | Displays the current system date and time. |
| `whoami` | Displays the current logged-in user. |
| `pwd` | Displays the present working directory. |
| `ls` | Lists files and directories. |
| `mkdir` | Creates a new directory. |
| `cd` | Changes the current directory. |
| `touch` | Creates a new empty file. |
| `cat` | Displays the contents of a file. |
| `chmod` | Changes file permissions. |
| `chown` | Changes the owner of a file or directory. |
| `whereis` | Displays the location of Linux commands. |
