# DAY-9
Introduction to HTML
## Date
13 July 2026
### Objective
To understand the basics of HTML, learn how web browsers render HTML documents, and create simple webpages using Notepad.
### Topics Covered
### 1. Introduction to HTML
- Learned that **HTML (HyperText Markup Language)** is the standard markup language used to create web pages.
- Understood that HTML uses tags to define the structure and content of a webpage.
- Learned the purpose of opening and closing tags.
### 2. What is a Web Browser?
Studied the role of a web browser in displaying web pages.
Examples of web browsers:
- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Safari
- Opera
Learned that browsers read HTML files and display them in a user-friendly webpage format.
### 3. How a Browser Renders an HTML Document
Learned the process followed by a browser:
1. The browser opens the HTML file.
2. It reads the HTML code line by line.
3. It creates the page structure.
4. CSS (if present) styles the webpage.
5. JavaScript (if present) adds interactivity.
6. Finally, the webpage is displayed on the screen.
### 4. DOM (Document Object Model)
Studied the concept of the **Document Object Model (DOM).**
- Learned that every HTML document is converted into a tree-like structure.
- Understood that JavaScript can access and modify HTML elements through the DOM.
### 5. Structure of an HTML Document
Learned the basic structure of an HTML page.
Main elements studied:
- `<!DOCTYPE html>`
- `<html>`
- `<head>`
- `<title>`
- `<body>`
Understood the purpose of each section in creating a valid HTML document.
### 6. HTML Heading Tags
Learned the six heading tags:
- `<h1>`
- `<h2>`
- `<h3>`
- `<h4>`
- `<h5>`
- `<h6>`
Understood that `<h1>` is the largest heading and `<h6>` is the smallest.
### 7. HTML Paragraph Tag
Learned the use of the `<p>` tag for writing paragraphs and displaying text content on a webpage.
### 8. HTML Lists
Studied different types of lists:
- Ordered List (`<ol>`)
- Unordered List (`<ul>`)
- List Item (`<li>`)
Learned how lists help organize information on a webpage.
### 9. HTML Links
Learned how hyperlinks are created using the `<a>` tag.
Understood:
- `href` attribute
- Opening web pages through hyperlinks
- Linking one webpage to another
### 10. HTML Images
Learned how to display images using the `<img>` tag.
Studied commonly used attributes:
- `src`
- `alt`
- `width`
- `height`
Also learned that the image file should be stored in the correct folder so that it displays properly.
### Practical Work Performed
During today's lab session, basic HTML programs were created using **Notepad**.
The practical activities included:
- Created the first HTML webpage.
- Saved the file with the **.html** extension.
- Opened the webpage in a web browser.
- Created a webpage displaying **"Hello Students"** as the main heading.
- Created a **Personal Information** webpage using headings and paragraphs.
- Practiced adding headings, paragraphs, links, images, and basic HTML elements.
- Learned how changes in the HTML code are reflected after refreshing the browser.
### Software Used
- Notepad
- Google Chrome / Microsoft Edge
### Learning Outcomes
- Understood the basics of HTML.
- Learned the structure of an HTML document.
- Gained knowledge of how web browsers render HTML pages.
- Learned the concept of the Document Object Model (DOM).
- Created simple webpages using Notepad.
- Practiced heading tags, paragraph tags, lists, links, and images.
- Learned how to save and execute HTML files in a web browser.
### Conclusion
Day 9 introduced the fundamentals of HTML and web development. Along with learning the theory behind HTML, web browsers, DOM, and webpage structure, basic webpages were successfully created using Notepad and executed in a web browser. The session provided a strong foundation for developing webpages and understanding how HTML documents are rendered, preparing us for more advanced HTML and web development concepts in future sessions.
