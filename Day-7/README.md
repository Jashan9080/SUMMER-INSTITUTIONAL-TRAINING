# DAY-7
Networking Basics
## Date
09 July 2026
### Objective
To understand the fundamental concepts of computer networking, including IP addressing, subnetting, DNS, DHCP, MAC addressing, network cables, connectors, crimping, and cable testing.
### Topics Covered
### 1. IP Addressing
- Learned the purpose of IP addresses in a network.
- Understood the difference between Public and Private IP addresses.
- Studied IPv4 addressing basics.
### 2. Subnetting
- Learned the concept of dividing a network into smaller subnetworks.
- Understood the importance of subnet masks in network communication.
### 3. Default Gateway
- Learned that the default gateway connects a local network to external networks.
- Understood its role in sending data outside the local network.
### 4. MAC Address
- Studied the unique physical address assigned to every network interface.
- Learned its importance in communication within a Local Area Network (LAN).
### 5. DNS (Domain Name System)
- Learned how DNS converts domain names into IP addresses.
- Understood why DNS is essential for accessing websites.
### 6. DHCP (Dynamic Host Configuration Protocol)
- Learned how DHCP automatically assigns:
  - IP Address
  - Subnet Mask
  - Default Gateway
  - DNS Server
- Understood the advantages of automatic IP configuration.
### 7. Types of Network Cables and Connectors
- Straight-through Cable
- Crossover Cable
- RJ45 Connector
Learned where each cable type is commonly used in networking.
### 8. Crimping Basics
- Learned the purpose of crimping Ethernet cables.
- Studied the correct wire color sequence used while attaching RJ45 connectors.
### 9. Cable Testing Basics
- Learned how a cable tester is used.
- Understood how cable testing identifies wiring faults and verifies proper cable connections.
### Practical Work
No practical work was performed during today's session.
The complete session focused on understanding networking concepts and writing notes in the practical file.
### Learning Outcomes
- Gained knowledge of IP Addressing and Subnetting.
- Understood the working of DNS and DHCP.
- Learned the importance of MAC Address and Default Gateway.
- Studied different Ethernet cable types and RJ45 connectors.
- Understood the basics of crimping and cable testing
### Conclusion
Day 7 mainly focused on strengthening theoretical knowledge of computer networking. Although no hands-on practical activity was conducted, the session provided a clear understanding of important networking concepts such as IP addressing, subnetting, DNS, DHCP, MAC addressing, Ethernet cables, RJ45 connectors, crimping, and cable testing. These concepts will serve as the foundation for future networking practicals and troubleshooting tasks.
