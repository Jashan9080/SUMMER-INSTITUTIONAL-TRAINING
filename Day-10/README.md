# DAY-10
HTML Webpage Development and Source Code Analysis
## Date
14 July 2026
### Objective
To understand how real-world websites are structured by viewing their HTML source code and to develop simple webpages using only HTML. The session also focused on practicing HTML by creating a custom webpage and documenting the output in the practical file.
### Activities Performed
### 1. Understanding Website Source Code
Today's session began with exploring the HTML source code of existing websites.
### Practical Activity
- Opened a website (such as Netflix) in a web browser.
- Used **Right Click → View Page Source** to view the website's HTML source code.
- Observed how HTML elements are used to structure a webpage.
- Understood that modern websites also use CSS and JavaScript along with HTML.
### Learning Outcomes
- Learned how browsers display the source code of a webpage.
- Understood the basic structure of a real-world website.
- Observed the use of HTML tags inside professional websites.
### 2. HTML Webpage Development
Created a custom webpage using only HTML.
### Practical Activity
- Created a new HTML file using **Notepad**.
- Saved the file with the **.html** extension.
- Opened the webpage in a web browser.
- Added headings, paragraphs, images, and text using HTML tags.
- Modified the webpage and refreshed the browser to view changes.
### 3 Creating a Netflix-Themed Webpage
As part of the practical session, I created a simple **Netflix-inspired webpage** using HTML.
### Features Included
- Netflix heading/logo text.
- Trending Movies section.
- Hollywood Movies section.
- Bollywood Movies section.
- Multiple movie posters displayed using HTML image tags.
- Proper headings and page layout using basic HTML elements.

### Image Handling

- Downloaded movie poster images.
- Saved all images in the **same folder** as the HTML file.
- Used the `<img>` tag with the correct **src** attribute.
- Displayed images successfully on the webpage.
This activity helped me understand how local image files are linked with HTML webpages.
### 4. Practical File Work
The output of the HTML webpages created during the practical session was printed/pasted in the practical file as instructed.
### 5. Assignment Work
The faculty assigned HTML-related questions based on today's practical session.
The assignment included:
- Basic HTML programming questions.
- Writing HTML code.
- Understanding webpage structure.
- Completing the assigned work in the notebook/file.
### Software Used
- Notepad
- Google Chrome / Microsoft Edge
### Skills Practiced
- Viewing webpage source code.
- Creating HTML files.
- Saving files with the **.html** extension.
- Using HTML headings and paragraphs.
- Displaying images using the `<img>` tag.
- Linking local image files with webpages.
- Organizing webpage content using HTML.
- Refreshing webpages after making code changes.
### Learning Outcomes
By the end of today's session, I learned:
- How to view the source code of an existing website.
- Basic understanding of webpage structure.
- How browsers interpret HTML code.
- How to create webpages using only HTML.
- How to insert local images into a webpage.
- Importance of storing images in the same folder as the HTML file.
- Practical experience in designing a simple themed webpage.
- Improved confidence in writing and testing HTML code.
### Conclusion
Day 10 was a fully practical session focused on applying HTML concepts through real-world examples. By analyzing the source code of an existing website and creating a custom Netflix-themed webpage using only HTML, I gained valuable hands-on experience in webpage development. Working with headings, images, and local files enhanced my understanding of HTML and provided a strong foundation for future web development topics such as CSS and JavaScript.
