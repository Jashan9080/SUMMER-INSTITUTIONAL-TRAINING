#  DAY-6 
Networking Practical and Computer Hardware Exploration
## Date
08 July 2026
## Department
Computer Science & Engineering (CSE)

### Objective
The objective of today's training session was to gain practical knowledge of **Ethernet cable wiring**, understand the internal components of a computer system, and strengthen our knowledge of computer hardware through hands-on learning. We also completed the pending practical file work in the laboratory.
### Topics Covered
### 1. Ethernet Cable Wiring
Today's practical session focused on preparing an Ethernet (LAN) cable using an RJ-45 connector.
### Activities Performed
- Learned the structure of an Ethernet cable.
- Identified the eight internal wires.
- Arranged the wires in the correct colour sequence.
- Inserted the wires into the RJ-45 connector.
- Understood the importance of maintaining the correct wiring standard.
### Standard Colour Sequence (T568B)
1. White-Orange
2. Orange
3. White-Green
4. Blue
5. White-Blue
6. Green
7. White-Brown
8. Brown
#### Learning Outcomes
- Understanding Ethernet cable construction.
- Learning the correct colour coding.
- Basic networking cable preparation.
- Importance of proper cable termination.
### 2. Computer Hardware Exploration
A faculty member explained the internal components of the **System Unit (CPU Cabinet)** and demonstrated how different hardware components work together inside a computer.
### Hardware Components Studied
- Motherboard
- Processor (CPU)
- RAM
- ROM
- HDD (Hard Disk Drive)
- SSD (Solid State Drive)
- SMPS (Power Supply Unit)
- Heat Sink
- Cooling Fan
- CMOS Battery
- BIOS/UEFI Chip
- SATA Cables
- Power Cables
- Expansion Slots
- PCIe Slots
- USB Headers
- Network Interface Components
### Concepts Learned
- Working of RAM and temporary memory.
- Difference between HDD and SSD.
- Function of the processor.
- Data flow inside the computer.
- Purpose of different cables and connectors.
- Role of the motherboard in connecting all hardware components
### 3. Safe Mode in Windows
We also learned about **Safe Mode** and its importance in troubleshooting Windows operating systems.
### Topics Covered
- What is Safe Mode?
- Purpose of Safe Mode.
- Different types of Safe Mode.
- Situations where Safe Mode is used.
- How Safe Mode helps in diagnosing software and driver-related problems.
### Types of Safe Mode
- Safe Mode
- Safe Mode with Networking
- Safe Mode with Command Prompt
### Learning Outcomes
- Understanding the purpose of Safe Mode.
- Learning when to use Safe Mode during system failures.
- Identifying software, driver, and startup issues using Safe Mode.
- Performing basic troubleshooting in a minimal Windows environment.
